Purpose of this document
The purpose of this document is to explain what the project will have to offer, representing it as a black box and describing its external behavior.
General description of the project
The main objective of this project is the research and analysis of existing tools for the creation of an interactive dashboard that allows forensic analysis of file properties, jumplist, .LNK file properties, USB device properties, Browser history, shellbags , shimcache, appcache, prefetch and superfetch.
